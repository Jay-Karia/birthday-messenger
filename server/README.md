# Server

## Setup

**Install dependencies**

`pip install -r requirements.txt`

**Set up environment variables in a .env file:**
```
EMAIL_USER
SENDGRID_API_KEY
GROQ_API_KEY
WEBHOOK_VERIFY_TOKEN
GRAPH_API_TOKEN
PORT
WHATSAPP_PHONE_NUMBER_ID
AUTH_USER
AUTH_PASS
APP_SECRET
CERTIFICATE_PASSWORD
```

**Run the server:**

`python main.py`
